# 50 Agent结果

## 实验用途

50人实验用于建立和检查最小行为闭环：天气与活动需求影响出行，Agent在步行、公交和网约车之间选择；网约车采用S1/S2车辆守恒；首次失败后允许一次fallback；公交和网约车共享道路。年龄、就业状态、数字接入和家庭协助影响可用方式及选择，但不直接指定最终方式。

这一规模主要回答“机制能否运行”，不用于寻找人口阈值或预测上海政策效果。

## 优惠券政策

- `C0_no_coupon`：无券基线；
- `C1_public_limited`：全年龄有限公共抢券；
- `C2_elder_limited`：老年定向有限发券；
- `C3_mixed`：70%公共池和30%老年保留池，并允许有限社区/电话代理渠道。

所有券统一八折，每人每天最多获得并核销一张。年轻人公共券参与概率高于老人；非数字且无家庭协助的老人不能独立叫车。主实验关闭网约车非运力随机失败，使派单失败主要来自真实车辆约束。

## 参数与结果

- 人数：50；
- seeds：3001–3030；
- 每日券池：10张；
- 文件：`policy_summary.csv`和`experiment_metadata.json`。

结果用于和200人实验比较规模放大前后的需求、等待、失败、道路压力、必要活动完成与热风险，不应和200人结果合并求平均。
