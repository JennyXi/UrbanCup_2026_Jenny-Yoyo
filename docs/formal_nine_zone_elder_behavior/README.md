# 九区模型中的稳定老年出行机制

本目录说明从 `experiment` 分支筛选进入 `main` 的老年出行行为机制。参数来自简化实验的稳定性确认，用于机制模拟，不是上海实测估计值，也不应解释为现实预测。

## 正式候选参数

| 参数 | 值 | 作用范围 |
|---|---:|---|
| 60+ 网约车方式常数 | 0.3 | 进入方式效用，表示老年人的网约车倾向不再被过度压低 |
| 60+ 恶劣天气户外暴露权重 | 1.6 | 放大高温、暴雨下户外分钟的感知负效用 |
| 老年必要出行条件费用敏感系数 | 0.9 | 仅在 W1/W2 天气窗口内的 work、medical 出行中使用 |
| 老年公交—地铁换乘感知负担 | 3 分钟/次 | 只增加感知时间，不改变实际运行时间 |
| 老年 medical need 权重 | low 1.0 / standard 1.2 / high 1.5 | 只放大老年医疗需要者的天气暴露感知负效用 |

## 作用链条

Agent 仍在可用方式中比较门到门时间、费用、迟到、天气暴露、换乘负担和随机效用，然后选择效用最高的方式。上述参数不会直接指定 Agent 打车，也不会修改目的地或活动目的。

数字接入规则保持不变：数字接入者可独立请求网约车；非数字接入但有家庭协助者可由家人代理；非数字且无家庭协助者不能独立请求网约车。

## 边界

- 条件费用敏感系数改变的是 Agent 对费用的感知权重，不改变实际车费。
- 换乘负担改变的是选择效用，不改变实际出行时间。
- medical need 权重只用于方式选择阶段，不改变生理热剂量公式。
- 本次合入不包含敏感性候选配置、优惠券政策脚本或实验输出。
